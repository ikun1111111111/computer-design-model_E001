## 数据模型设计

### ER 图
```
┌─────────────────┐
│      User       │
│─────────────────│
│ id (PK)         │
│ name            │
│ level           │
│ created_at      │
└────────┬────────┘
         │
         ├──────────────────────┐
         │                      │
         ▼                      ▼
┌─────────────────┐    ┌─────────────────┐
│  PracticeLog    │    │    Portfolio    │
│─────────────────│    │─────────────────│
│ id (PK)         │    │ id (PK)         │
│ user_id (FK)    │    │ user_id (FK)    │
│ craft_id        │    │ image_url       │
│ craft_name      │    │ craft_type      │
│ duration        │    │ description     │
│ score           │    │ created_at      │
│ created_at      │    └─────────────────┘
└─────────────────┘
```

### 模型特点
1. **外键级联删除**: 删除用户时自动删除关联数据
2. **索引优化**: 常用查询字段已创建索引
3. **中文注释**: 所有字段都有 COMMENT
4. **时间戳**: 自动记录创建时间
5. **UTF8MB4**: 支持完整 Unicode 字符集


```bash

```

```bash
cd backend

```

### 4. 运行完整测试（需要 MySQL）
