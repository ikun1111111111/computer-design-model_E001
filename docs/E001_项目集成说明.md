## 三、快速开始

### 1. 安装依赖
```bash
cd backend
pip install -r requirements.txt
```

### 2. 配置数据库
编辑 `.env` 文件或设置环境变量：
```bash
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=your_password
MYSQL_DATABASE=inheritor_db
```

### 3. 执行数据库迁移
```bash
cd backend
python app/db/migrations/001_create_user_tables.py
```

### 4. 运行模型测试（无需数据库）
```bash
cd backend
python tests/test_e001_user_models_simple.py
```

### 5. 启动后端服务
```bash
cd backend
python run.py
```

### 6. 运行 API 测试（需要服务运行）
```bash
cd backend
python tests/test_e001_api.py
```

## 四、API 使用示例

### 创建用户
```bash
curl -X POST "http://localhost:8000/api/v1/users?name=张三&level=1"
```

### 创建学习记录
```bash
curl -X POST "http://localhost:8000/api/v1/users/1/practice-logs?craft_id=101&craft_name=苏绣&duration=1800&score=85.5"
```

### 获取用户完整档案
```bash
curl "http://localhost:8000/api/v1/users/1/profile"
```

## 五、数据模型关系

```
┌─────────────────┐
│      User       │
│─────────────────│
│ id (PK)         │
│ name            │
│ level           │
│ created_at      │
└────────┬────────┘
         │
         ├──────────────────────┐
         │                      │
         ▼                      ▼
┌─────────────────┐    ┌─────────────────┐
│  PracticeLog    │    │    Portfolio    │
│─────────────────│    │─────────────────│
│ id (PK)         │    │ id (PK)         │
│ user_id (FK)    │    │ user_id (FK)    │
│ craft_id        │    │ image_url       │
│ craft_name      │    │ craft_type      │
│ duration        │    │ description     │
│ score           │    │ created_at      │
│ created_at      │    └─────────────────┘
└─────────────────┘
```

**特性**:
- 外键约束：保证数据完整性
- 级联删除：删除用户自动清理关联数据
- 索引优化：常用查询字段已建索引
- 中文注释：所有字段都有 COMMENT
